# PRIMERS/PUBLICATIONS Primer Booklet Series Plan

## Objective
Build the `PRIMERS/PUBLICATIONS` primer booklet series with beginner-friendly introduction prompts.

## Scope
- Create 10 booklets.
- Each booklet includes one `01_introduction.md` section prompt.
- Preserve both a short question and long-form guidance question.
- Maintain JSON representation of booklet metadata/content.

## Rules
- Ignore `model`.
- Ignore `temperature`.
- Preserve specific instructional constraints in each long-form question.
