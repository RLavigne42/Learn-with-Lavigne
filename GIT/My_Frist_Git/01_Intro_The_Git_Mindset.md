# 01 — The Git Mindset

## Section Goal
Understand Git's three states:
- Working Directory
- Staging Area
- Repository

## Command

### `git gui` (Visual Learning)
- `N/A` — Opens Git's default GUI so beginners can see files move into the staging area.

## Quick Win
Use `git gui` once before CLI-heavy work to build intuition for staging vs committing.
