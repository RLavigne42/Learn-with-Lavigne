### Curriculum Synopsis (High-Level)

- **00 Index:** framing, architecture, and pedagogical pathways.
- **01 Mindset:** Three Trees model, distributed version control, Git vs GitHub.
- **02 Setup:** identity, `init.defaultBranch`, aliases, `core.autocrlf`, SSH.
- **03 Local Workflow:** `init/clone`, `status`, `diff`, `add`, `commit`, clean history habits.
- **04 Branching/Merging:** parallel work, merge strategy, conflict resolution.
- **05 Remotes:** remote topology, upstream tracking, `fetch` vs `pull`, prune behavior.
- **06 Undo/Recovery:** `restore`, `reset` variants, `revert`, and `reflog` safety net.
- **07 Utility:** `stash`, semantic `tag`s, `.gitignore` hygiene.
- **08 Diagnostics:** `bisect`, `annotate`, `rebase -i`, `cherry-pick`.
- **09 Appendix:** consolidated CLI cheat sheet and architectural recap.
