## Command Synopsis
- `git --version` — Check installed version
- `git version` — Display version information about Git
- `git help` — Display help information about Git
- `update-git-for-windows` — Update Git for Windows specifically

## Fundamental Usage
### `git --version`
- Purpose: Check installed version
- Core patterns:
```bash
git --version
git version
```

### `git version`
- Purpose: Display version information about Git
- Core patterns:
```bash
git version
```

### `git help`
- Purpose: Display help information about Git
- Core patterns:
```bash
git help
git help commit
```

### `update-git-for-windows`
- Purpose: Update Git for Windows specifically
- Core patterns:
```bash
update-git-for-windows
```
