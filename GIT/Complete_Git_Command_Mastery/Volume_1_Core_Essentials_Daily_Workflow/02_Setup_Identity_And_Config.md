# Setup Identity & Configuration

**Volume 1: Core Essentials & Daily Workflow**

## Focus Topics
- Global vs Local Config
- Credentials
- Variables

## Command Synopsis
- `git config` — Get and set repository or global options
- `git credential` — Retrieve and store user credentials
- `credential-manager` — External credential helper
- `git var` — Show a Git logical variable

## Fundamental Usage
### `git config`
- Purpose: Get and set repository or global options
- Core patterns:
```bash
git config --help
git config
```

### `git credential`
- Purpose: Retrieve and store user credentials
- Core patterns:
```bash
git credential --help
git credential
```

### `credential-manager`
- Purpose: External credential helper
- Core patterns:
```bash
credential-manager --help
credential-manager
```

### `git var`
- Purpose: Show a Git logical variable
- Core patterns:
```bash
git var --help
git var
```
