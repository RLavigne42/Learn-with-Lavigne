## Command Synopsis
- `git` — Main command entry point (structure overview)
- `git gui` — A portable graphical interface to Git
- `gitk` — The Git repository browser

## Fundamental Usage
### `git`
- Purpose: Main command entry point (structure overview)
- Core patterns:
```bash
git
git help
```

### `git gui`
- Purpose: A portable graphical interface to Git
- Core patterns:
```bash
git gui
git gui --help
```

### `gitk`
- Purpose: The Git repository browser
- Core patterns:
```bash
gitk
gitk --all
```
