# Complex Workspaces & Large Repos

**Volume 2: Advanced Management, Auditing & Repair**

## Focus Topics
- Submodules
- Worktrees
- Large Files

## Command Synopsis
- `git submodule` — Initialize, update or inspect submodules
- `git worktree` — Manage multiple working trees
- `git sparse-checkout` — Reduce your working tree to a subset of files
- `git lfs` — Git Large File Storage
- `git scalar` — Tool for managing large Git repositories
