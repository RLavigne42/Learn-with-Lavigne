## Command Synopsis
- `git archive` — Create an archive of files from a named tree
- `git bundle` — Move objects and refs by archive
- `git fast-export` — Git data exporter
- `git fast-import` — Backend for fast Git data importers

## Fundamental Usage
### `git archive`
