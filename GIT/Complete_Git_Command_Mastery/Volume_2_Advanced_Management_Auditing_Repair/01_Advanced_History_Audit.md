## Command Synopsis
- `git blame` — Show what revision and author last modified each line
- `git annotate` — Annotate file lines with commit information
- `git reflog` — Manage reflog information (History of HEAD)
- `git whatchanged` — Show logs with differences each commit introduces
- `git show` — Show various types of objects (commits, trees, blobs)
