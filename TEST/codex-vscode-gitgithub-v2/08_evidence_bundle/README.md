# Evidence Bundle Guide

Store all objective proof for Module 08 here.

Recommended contents:
- `commands.log` (command transcript)
- `validation.md` (test/lint/manual outcomes)
- `conflict-case.md` (before/after and rationale)
- `recovery-case.md` (reflog/recovery steps)
- `pr-links.md` (issue, PR, review thread references)

Use UTC timestamps and short interpretation notes for each artifact.
